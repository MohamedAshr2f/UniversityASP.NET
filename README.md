**University Project is a robust University Management System backend built with ASP.NET Core Web API**. It is designed using Clean Architecture principles and a Code-First approach to ensure scalability, maintainability, and testability.

This project implements advanced design patterns such as CQRS (Command Query Responsibility Segregation) and the Mediator pattern to decouple message sending and handling. It features a comprehensive security system using Identity, JWT, and role-based authorization, making it suitable for managing Students, Instructors, Departments, and complex University workflows.
